# Mohammed Latif Siddiq's Personal Website

This is the source code for the personal academic website of Mohammed Latif Siddiq.

## Project Structure

- `index.html`: The main homepage.
- `css/`: Stylesheets. `main.css` contains custom styles.
- `js/`: JavaScript files. `main.js` contains custom logic.
- `img/`: Images used on the site.
- `public/`: Public assets.
- `favicons/`: Favicon resources.

## Tech Stack

- **HTML5**
- **CSS3**: Custom styles + Bootstrap 3
- **JavaScript**: jQuery + Custom scripts

## Local Development

To run this website locally, you can use any static file server.

### Using Python

```bash
# Python 3
python3 -m http.server 8000
```

Then open [http://localhost:8000](http://localhost:8000) in your browser.

### Using Node.js (http-server)

```bash
npx http-server .
```
